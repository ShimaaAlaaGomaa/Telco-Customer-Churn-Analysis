# Telco Customer Churn Analysis - Power BI Dashboard

## Project Overview
This project focuses on analyzing customer churn for a telecommunications company. The goal is to identify the key factors that lead to customer attrition and provide actionable insights to improve customer retention.

### 3. Key Drivers of Churn

## Key Insights
* **Churn by Contract:** Customers with **Month-to-Month** contracts show significantly higher churn rates compared to long-term contracts.
* **Payment Methods:** Electronic check users have a higher tendency to churn.
* **Tenure:** New customers (0-1 year) are at the highest risk of leaving.
* **Service Factors:** Identification of specific internet services that impact retention.

## Tools Used
* **Power BI Desktop:** For data visualization and reporting.
* **DAX:** Used for creating custom metrics and KPI calculations.
