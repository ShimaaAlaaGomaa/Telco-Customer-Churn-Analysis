# Telco Customer Churn Analysis

## 📌 Introduction
Customer churn is one of the most critical metrics for telecommunications companies. This project provides a deep dive into customer behavior to understand why customers leave and how to improve loyalty using data-driven insights.

## 📝 Project Overview
This project involves a complete end-to-end data analysis lifecycle, starting from raw data cleaning using Python to creating an interactive dashboard in Power BI. It analyzes various factors such as contract types, payment methods, and customer tenure to predict and visualize churn patterns.

## 📊 Project Info
* **Status:** Completed
* **Level:** Professional Portfolio Project
* **Field:** Data Analytics / Business Intelligence

## 🎯 Objective
The primary goal of this project is to:
* Identify the key drivers behind customer churn.
* Visualize churn distribution across different demographics and services.
* Provide actionable recommendations to management for reducing attrition rates.

## 🔍 Scope
* **Data Cleaning & Preprocessing:** Handling missing values and formatting data for analysis.
* **Exploratory Data Analysis (EDA):** Uncovering hidden patterns and correlations using Python.
* **Dashboard Development:** Creating a user-friendly interface to track KPIs.
* **Statistical Analysis:** Identifying feature importance for churn prediction.

## 🛠️ Tools & Technologies
* **Python:** Used for Data Cleaning, Preprocessing, and EDA (Pandas, NumPy, Matplotlib/Seaborn).
* **Power BI:** Used for Data Modeling, DAX calculations, and interactive Visualizations.
* **Excel/CSV:** Source data format.

## 💡 Key Insights
* **Contract Influence:** Customers on month-to-month contracts are significantly more likely to churn than those on yearly plans.
* **Financial Drivers:** High monthly charges and electronic check payments correlate with higher churn rates.
* **Tenure Impact:** The first 12 months are the most critical period for customer retention.
* **Tech Support:** Customers without technical support services show a higher tendency to leave.

## 🚀 Recommendations
* **Incentivize Long-term Contracts:** Offer discounts or benefits for customers switching from monthly to yearly plans.
* **Target New Customers:** Implement a loyalty program specifically for customers in their first year.
* **Improve Support Services:** Promote tech support packages to high-risk segments to increase satisfaction.
* **Payment Optimization:** Encourage automated payment methods to reduce friction in the billing process.
